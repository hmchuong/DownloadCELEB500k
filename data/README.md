# DATA
There are two folder inside:
- `images`: containing downloaded images
- `urls`: containing url files

The structure of `data` folder 
```
images
|--------<url folder 1>....
            | --- <Identity folder 1>
                          |---- <image 1>
                          |---- ...
                          |---- <image n>
            | --- ...
            | --- <Identity folder n>
|-------- ...
|-------- <url folder n>


urls
|--------<url folder 1>....
            | --- <Identity text file 1>
            | --- ...
            | --- <Identity text file n>
|-------- ...
|--------<url folder n>....
```
